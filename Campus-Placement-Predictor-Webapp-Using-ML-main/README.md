# 🎓 Job Admission & Placement Predictor

A Machine Learning based web application that predicts **Campus Placement Probability** and **Higher Education Admission Chances** based on a student's academic profile.

This system helps students understand their career opportunities by analyzing factors like **CGPA, internships, projects, skills, and academic background**.

---

# 🚀 Features

✔ Predict Campus Placement Probability  
✔ Predict Higher Education Admission Chances  
✔ Interactive Student Profile Form  
✔ Dashboard showing prediction results  
✔ Data Visualization with charts  
✔ Clean and user-friendly interface  

---

# 🧠 Machine Learning Models

The system uses Machine Learning algorithms to analyze student academic data and predict outcomes.

Models used:

- Random Forest Classifier → Placement Prediction
- Logistic Regression → Admission Probability
